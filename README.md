# **Ema Jhon**

![alt text](https://www.dl.dropboxusercontent.com/s/3vzfttnx24xb1ik/emajhon.png?dl=0)

Ema John is a simple ecommerce website where users can buy their products and pay with stripe payment gateway and admins can manage their products.
<br />
[Live website](https://ema-john-simple-ishad.web.app)

---

## **Language & technologies used**
* JavaScript
* ReactJS
* NodeJS
* MongoDB
* Firebase
* Heroku

---

## **Packeges used**

#### Frontend

* Stripe JS

* Nodemon
